# sandbox-react-ts
Test project with React and TypeScript implementation

## Requirements

* Node.js v8.x

## Stack

* FuseBox
* TypeScript
* React
* {} - state manager
* Styled-Components
* Jest

## Installation

`npm install`

## Run

`npm start`

And open: http://localhost:4445/

## Assets

* Characters assets were taken from https://sarge.itch.io/vigil Indie Point & Click Adventure
